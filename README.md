  ## Installation
  Make sure to install node by typing:
   ```
   npm i
   ```
  Create database by typing
  ```
  source db/schema.sql
  ```
  and then seed the database using
  ```
  node seeds/index.js
  ```
  then you can start the server by typing:
  ```
  node server.js
  ```
